# CKEditor Quick Reference for Admin Users

## Getting Started

When editing content in the admin panel, you'll see a rich text editor with formatting options.

## Using the Toolbar

### Text Formatting
| Button | Shortcut | Function |
|--------|----------|----------|
| **B** | Ctrl+B | Make text **bold** |
| **I** | Ctrl+I | Make text *italic* |
| **U** | Ctrl+U | <u>Underline</u> text |

### Headings
Click the **Heading** dropdown to select:
- Heading 1 (H1) - Main title
- Heading 2 (H2) - Subheading
- Heading 3-6 - Further subdivisions

### Lists
- **Bullet List** - Create unordered (•) lists
- **Numbered List** - Create ordered (1, 2, 3...) lists

### Other Formatting
- **Block Quote** - Highlight important text
- **Alignment** - Align text left, center, right, or justify
- **Link** - Add hyperlinks
- **Undo/Redo** - Reverse or repeat actions

## Examples

### Creating a List
1. Type your first item
2. Press Enter
3. Click the Bullet List or Numbered List button
4. Type items (press Enter between items)
5. Press Enter twice to end the list

### Adding a Link
1. Select the text you want to link
2. Click the Link button
3. Enter the URL (e.g., https://example.com)
4. Press Enter or click outside

**Note**: Links automatically open in a new tab for safety.

### Using Headings
```
Main Title (H1)
  Subheading (H2)
    Details (H3)
```

## Content Limits

Each field has a maximum word count:
- 💬 Service Description: 500 words
- 📝 FAQ Answer: 300 words
- 💝 Testimonial Quote: 200 words
- 🎯 Feature Description: 300 words
- 📱 Banner Subtitle: 100 words

If you exceed the limit, you'll see an error message.

## What You Can Do
✅ Bold, italic, underline text
✅ Add headings and subheadings
✅ Create bullet and numbered lists
✅ Add blockquotes
✅ Insert links
✅ Align text

## What You Can't Do
❌ Upload images (safe design)
❌ Embed videos
❌ Add scripts or code
❌ Use CSS styling
❌ Use dangerous HTML tags

## Tips & Tricks

### Professional Formatting
1. Use H1 for main topics only
2. Use H2 for sections
3. Use lists for multiple items
4. Use blockquotes for important notes

### Link Best Practices
- Use descriptive link text (not "Click here")
- Include https:// in URLs
- Links open in new tabs automatically

### Content Tips
- Keep it concise
- Use headings to organize
- Use lists for clarity
- Break up long paragraphs

## Troubleshooting

**The editor looks broken?**
- Refresh the page
- Clear your browser cache
- Try a different browser

**My formatting disappeared?**
- Check word count limits
- Try saving again
- Contact support if issue persists

**Links don't work?**
- Make sure URL starts with http:// or https://
- Check the URL is correct
- Test in a new tab

## Support
If you encounter issues, contact your system administrator.
